## Run
```
cd P2P-Video
docker-compose up --build
```

# Endpoints
`http://0.0.0.0:8000/upload-file` - form to upload file\
`http://0.0.0.0:8000/download-file/` - form to download file\
`http://0.0.0.0:5001/webui` - IPFS web GUI
